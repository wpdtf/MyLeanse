# MyLeanse

**Бот для работы с контактными линзами**

---

## Описание

MyLeanse позволяет отслеживать время ношения контактных линз. С его помощью вы можете:  
- Отмечать время, когда линзы были надеты.  
- Отмечать время, когда линзы были сняты.  
- Собирать статистику по общему времени использования линз.

---

## Особенности

- Хранение данных осуществляется в JSON-файле вместо полноценной базы данных.  
- Лёгкий сервис, подходящий для развертывания на слабых машинах.  
- Простое развертывание через Docker на Linux-виртуалке.

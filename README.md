# MyLeanse

Бот для работы с линзами

Текущая версия позволяет отмечать время когда линзы были надеты, а когда сняты, а после это информация позволит собрать статистику по общему времени использованию контактных линз.

Сервис для развертывания на очень слабой машине использует вместо базы данных хранение в json файле. Развертывается через docker на linux виртуалке.
